# STUDIO CYMLIN PUZZLE GAME

## Requirements:

- VS Code
- Python 3
- Pygame
- Pillow

## How to run:
``` 
python src/main.py
```

## How to play:
This is a two player cooperative game. One player will be able to control the cow, and the other controls the duck.

### Cow movement:

- Move up: Up Arrow
- Move down: Down Arrow    
- Move right: Right Arrow   
- Move left: Left Arrow    

### Duck movement:

- Move up: W Key
- Move down: S Key
- Move right: D Key
- Move left: A Key

### To restart if stuck:

- Press R Key

## Objective:

- The cow pushes the crate onto buttons to open fences.
- For locked boxes, the duck collects the key and uses it to unlock crates.
- Once the path is clear, the cow walks off the right edge to move to the next level.
- Press R if you need to restart the level.
- Press ESC to quit.
